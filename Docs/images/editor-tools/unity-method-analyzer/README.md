# Unity Method Analyzer Images

This directory should contain the following placeholder images for the Unity Method Analyzer documentation.

## Required Images

### Static Screenshots (PNG)

| Filename                 | Description                                           |
| ------------------------ | ----------------------------------------------------- |
| `analyzer-overview.png`  | Main window showing the analyzer with detected issues |
| `open-analyzer-menu.png` | The Tools menu showing how to open the analyzer       |
| `results-tree.png`       | Results tree view showing issues grouped by file      |
| `issue-detail-panel.png` | The detail panel showing full issue information       |
| `source-directories.png` | Source directories section with add/remove controls   |
| `severity-filter.png`    | Severity dropdown filter in action                    |
| `export-menu.png`        | Export dropdown menu showing all options              |

### Animated GIFs

| Filename             | Description                                       |
| -------------------- | ------------------------------------------------- |
| `first-scan.gif`     | Running the first analysis scan showing progress  |
| `grouping-modes.gif` | Switching between File/Severity/Category grouping |
| `search-filter.gif`  | Using the search filter to find specific issues   |

## Recommended Image Specifications

- **Screenshots**: 800-1200px wide, PNG format
- **GIFs**: 600-900px wide, optimized for file size
- **Theme**: Use Unity's default dark theme for consistency
